# Urdu-Hate-Speech-Detection---NUHONS-Dataset
This project benchmarks traditional and neural models for hate speech detection on the NUHONS dataset, focusing on Nastaliq Urdu YouTube comments. Key models include Naive Bayes, Logistic Regression, and various RNN architectures, with the best results from Logistic Regression (76.65% accuracy) and LSTM (64.25% multiclass F1-score).
